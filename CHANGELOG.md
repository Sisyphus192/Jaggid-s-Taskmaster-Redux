# Changelog

## [1.10-169] - 2026-05-02

First Redux release. The initial commit is Jaggid's untouched v1.10-168 source; every change since is a single bug fix.

### Fixed

- **TM-1** — `Craft` ActivitySet tooltip now mentions Cooking is included. The set already enabled `Cooking` (matching the in-game behavior), but the description only listed Assembling, Crafting, and Tailoring, which surprised players whose cooks abandoned crafts to make meals.
- **TM-2** — `Scavenge` ActivitySet tooltip typo fixed: `"Scrapping"` → `"Scavenging"`, and the trailing double period reduced to a single full stop.
